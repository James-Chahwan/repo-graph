User's framing: "this is like a map of a brain structure so that llm's use optimal graph context of memories to follow paths easier and quicker"

The core value prop is context savings — instead of an LLM reading thousands of files to understand a codebase, it navigates a pre-built graph to find exactly the files it needs for a given task. The graph acts as a cognitive shortcut.

**How to apply:** README and docs should lead with this pitch. The tool isn't just "code analysis" — it's an LLM context optimization layer. Frame features in terms of tokens saved and navigation speed.
